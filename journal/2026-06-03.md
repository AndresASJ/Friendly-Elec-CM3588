# 2026-06-03

## Lidarr/Soularr: stop the download flood — switch to request-only

Soularr was downloading hundreds of songs unprompted. Root cause: Hermes had been adding
artists to Lidarr with `monitor:"all"` + `monitorNewItems:"all"` + `searchForMissingAlbums:true`,
so every album of all 11 artists (Beatles, Pink Floyd, McCartney, Jackson, Chet Baker, …) was
marked **wanted** — **157 missing albums**. Soularr (`search_source = missing`, 5-min interval)
dutifully tried to grab the entire list from Soulseek every cycle.

Owner only wants the specific songs/albums they explicitly request. Fixes:

- **Lidarr reconfigured request-only:** unmonitored all **256** albums across the 11 artists and
  set every artist's `monitorNewItems = none`. Missing list **157 → 0**. (Artists themselves stay,
  just nothing is wanted until asked.)
- **Cleared the in-flight queue:** cancelled **281** queued Soulseek downloads via the slskd API.
- **Hermes behavior fixed** (`/opt/data/memories/USER.md`, Music section): new artists are now
  added with `monitor:"none"` + `monitorNewItems:"none"` + `searchForMissingAlbums:false`; then
  Hermes monitors **only the one requested album** and triggers `AlbumSearch` for just that. Big
  bold REQUEST-ONLY guardrail added. Restarted Hermes to reload.
- **slskd import-permission bug — FIXED:** slskd ran as **root**, so files it downloaded were
  `root:root` and Soularr/Lidarr (uid 1000) hit `Permission denied` on import → rollback. Fixed by
  running slskd as uid 1000: `chown -R 1000:1000 /mnt/drive1/AppData/slskd`, chown'd the leftover
  root-owned dirs under `/mnt/drive1/Downloads/Soulseek`, added `user: "1000:1000"` to the slskd
  service (manual `slskd-vpn` compose + repo `compose/slskd-gluetun.yml`), recreated **only** slskd
  (Gluetun/VPN left running). Verified: slskd `uid=1000`, logged into Soulseek as AndresASJ through
  the VPN, web API HTTP 200, Gluetun still healthy. (Compose backup: `.bak-perms-*`.)
- Note: slskd still shares **0 files** — low Soulseek upload priority can slow grabs; separate item.
